# Model Context Protocol (MCP)

**AI Engine** รองรับการทำงานร่วมกับ AI Agents (เช่น Cursor หรือ Claude Desktop) อย่างเต็มรูปแบบผ่านเทคโนโลยี **MCP** ซึ่งช่วยให้ AI สามารถเรียกใช้งาน API ของเรา (เช่น ดึงข้อความจาก PDF, ถอดเสียง, หรือค้นหาข้อมูลในเอกสาร) ได้โดยตรงผ่านการพิมพ์คำสั่งภาษาคน

ระบบ MCP ของเราให้บริการผ่านคลาวด์ (Heroku) ในรูปแบบ **SSE (Server-Sent Events)** ทำให้ผู้ใช้งานในทีมสามารถเชื่อมต่อได้ทันทีโดยไม่ต้องติดตั้งโค้ด Python ในเครื่องเลย

---

## 🚀 วิธีการเชื่อมต่อ (สำหรับ Cursor)

คุณสามารถตั้งค่าให้ Cursor รู้จักกับ AI Engine ได้ใน 1 นาที:

1. เปิด **Cursor** ไปที่ **Settings** (รูปเฟืองมุมขวาบน หรือกด `Cmd/Ctrl + Shift + J`)
2. ไปที่เมนู **Features** -> เลื่อนหาหัวข้อ **MCP Servers**
3. คลิกปุ่ม **+ Add New MCP Server**
4. กรอกข้อมูลการเชื่อมต่อดังนี้:
   - **Name:** `AI-Engine-Cloud` *(หรือชื่อใดก็ได้ตามต้องการ)*
   - **Type:** เปลี่ยนจาก `command` เป็น **`sse`**
   - **URL:** `https://ai-engine-mcp-5e80b6b703b9.herokuapp.com/sse`
5. กด **Save** และรอจนกว่าสถานะจะขึ้นไฟสีเขียว 🟢

## 🛠️ เครื่องมือที่ AI สามารถใช้งานได้ (Available Tools)

เมื่อเชื่อมต่อสำเร็จ AI ของคุณจะรู้จักเครื่องมือ (Tools) เหล่านี้และเรียกใช้เองโดยอัตโนมัติเมื่อจำเป็น:

| ชื่อเครื่องมือ | หน้าที่การทำงาน |
| :--- | :--- |
| `extract_text_ocr` | อ่านและดึงข้อความทั้งหมดจากไฟล์ PDF ในเครื่องของคุณผ่านระบบ OCR |
| `analyze_sheet_async` | อัปโหลดไฟล์ PDF เข้าสู่ระบบ AI Analysis เพื่อประมวลผล, สรุปเนื้อหา, และรับค่า `sheet_id` กลับมา |
| `transcribe_audio_async` | อัปโหลดไฟล์เสียง (.mp3, .wav) เพื่อถอดความและสรุปเนื้อหาอัตโนมัติ |
| `chat_with_rag` | พูดคุยหรือถาม-ตอบข้อมูลเฉพาะเจาะจงในเอกสาร โดยอ้างอิงจาก `sheet_id` ที่ประมวลผลไว้แล้ว |

## 💡 ตัวอย่างการสั่งงาน (Use Cases)

คุณสามารถเปิดช่องแชทใน Cursor (กด `Cmd/Ctrl + L`) และพิมพ์คำสั่งภาษาไทยผสมการใช้เครื่องมือได้เลยครับ:

### 1. ให้ AI ช่วยอ่านและสรุป PDF ทันที
> "ช่วยดึงข้อความจากไฟล์ `/Users/yourname/Downloads/document.pdf` ให้หน่อย แล้วเขียนสรุปประเด็นสำคัญมาเป็น Bullet points"

### 2. คอมโบ: วิเคราะห์เอกสาร + ถามตอบเชิงลึก (RAG)
การสั่งงานแบบคอมโบคือจุดเด่นของระบบนี้ AI จะทำการอัปโหลด รอผลลัพธ์ และนำไปค้นหาต่อให้เองทั้งหมด:
> "ช่วยเอาไฟล์ `/Users/yourname/Downloads/lecture1.pdf` ไปวิเคราะห์ในระบบ Analyze Sheet ให้หน่อย พอระบบประมวลผลเสร็จและได้ `sheet_id` มาแล้ว ให้ช่วยเอาไอดีนั้นไปถามในแชทให้ทีว่า **'อาจารย์เน้นย้ำเรื่องอะไรมากที่สุดในบทนี้?'**"

### 3. ผู้ช่วยถอดเสียงการประชุม
> "ฉันมีไฟล์เสียงสัมภาษณ์อยู่ที่ `/Users/yourname/audio/interview.mp3` ช่วยเอาไปเข้าสู่ระบบ Transcribe แล้วดึงเฉพาะสรุปผล (Summary) กลับมาให้ฉันดูหน่อย"
